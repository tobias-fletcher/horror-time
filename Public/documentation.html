Horror Movies

This API is a collection of horror movies.

You can make an account to store lists of your favorite movies

Business Logic URL Endoint HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None JSON object with data about all movies
Get information about a movie title /movies/:Title GET None JSON object with data about a specific movie with info about title, director, and year
Return data about director of movie by name /movies/Director/:name GET None JSON object with data about director
Allow new users to register /users POST JSON object with data of a new user containing username, password, email, date of birth JSON object with data about user that was addded including id
allow users to update their info /users/:Username PUT JSON object with updated information for Username, Email, Password JSON object with updated data
Allow users to add favorite movies /users/:Username/movies/:Title POST None JSON object with data about movies to be added
allow users to remove movies from favorites /users/:Username/movies/:Title DELETE None a message stating if movie was removed successfully
Allow users to deregister their account /users/:Username DELETE None a message stating if account was removed successfully
